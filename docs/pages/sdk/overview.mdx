# Enzyme SDK - Overview

The Enzyme SDK is based on [viem](https://viem.sh/). It facilitates making on-chain calls and sending transactions.

## Quick Start

To use the Enzyme SDK, add [@enzymefinance/sdk]() to your TypeScript project:

:::code-group

```bash [pnpm]
pnpm add @enzymefinance/sdk viem
```

```bash [yarn]
yarn add @enzymefinance/sdk viem
```

:::

### 1. Set up Viem

Create a public client (see the [viem documentation](https://viem.sh/docs/getting-started) for more details).

```ts twoslash
import { createPublicClient, http } from "viem";
import { mainnet } from "viem/chains";

const client = createPublicClient({ // [!code focus]
  chain: mainnet, // [!code focus]
  transport: http(), // [!code focus]
}); // [!code focus]
```

### 2. Use the Enzyme SDK

Use the public client to load on-chain data with the SDK.

```ts twoslash
import { createPublicClient, http } from "viem";
import { mainnet } from "viem/chains";
import { Vault } from "@enzymefinance/sdk"

const client = createPublicClient({ 
  chain: mainnet, 
  transport: http(), 
}); 

const name = await Vault.getName(client, { // [!code focus]
  vaultProxy: "0x1ce8aafb51e79f6bdc0ef2ebd6fd34b00620f6db" // [!code focus]
}); // [!code focus]
```
