# Depositor

The Enzyme API provides an endpoint to obtain data related to a depositor, including all their deposits into various vaults. Detailed documentation about request and response parameters is available in the Buf Schema registry.

- [GetDepositorTimeSeries](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetDepositorTimeSeries): timeseries of a depositor's performance data (e.g. aggregated NAV)
