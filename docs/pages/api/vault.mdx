# Vault

The Enzyme API provides various endpoints to obtain data related to a vault.

Detailed documentation about request and response parameters is available in the Buf Schema registry.

- [GetVault](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetVault): basic vault information
- [GetVaultActivities](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetVaultActivities): list of activities
- [GetVaultDepositors](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetVaultDepositors): list of depositors
- [GetVaultPortfolio](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetVaultPortfolio): current portfolio (assets and external positions)
- [GetVaultConfiguration](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetVaultConfiguration): vault configuration (fees and policies)
- [GetVaultTimeSeries](https://buf.build/avantgardefinance/enzyme/docs/main:enzyme.enzyme.v1#enzyme.enzyme.v1.EnzymeService.GetVaultTimeSeries): timeseries of vault performance data (e.g. NAV and share price)
